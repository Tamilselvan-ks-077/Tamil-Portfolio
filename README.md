# 🌌 Tamilselvan K.S — Personal Portfolio

An interactive, premium 3D developer portfolio showcasing full-stack capabilities, security engineering, and cyber reconnaissance projects. Built using React, Three.js (React Three Fiber), GSAP, Framer Motion, and Lenis for smooth momentum scrolling.

---

## 🚀 Key Features

*   **Interactive 3D Constellations**: Custom Three.js particle grids and web-shaped tech nodes that rotate and respond to cursor pointer hover.
*   **3D Device Mockups**: Interactive laptop, tablet, and phone mockups to preview responsive layouts of featured projects.
*   **Awwwards-Inspired Glassmorphic Design**: Clean UI/UX featuring high-end CSS blur backdrops, futuristic layout transitions, and subtle hover animations.
*   **Lenis Smooth Scroll & GSAP Animations**: Silk-smooth scrolling combined with ScrollTrigger revelations for dynamic parallax headers, timelines, and grids.
*   **Terminal/Command Palette**: A built-in spotlight search console (`Cmd/Ctrl + K`) to navigate links and trigger portfolio actions.

---

## 🛠️ Tech Stack

*   **Frontend Library**: React.js (v19)
*   **3D & Physics**: Three.js, `@react-three/fiber`, `@react-three/drei`
*   **Animations**: GSAP (GreenSock), Framer Motion
*   **Scroll Mechanics**: Lenis Smooth Scroll
*   **Module Bundler**: Vite (Fast HMR)
*   **Styles**: Vanilla CSS (Fluid flex/grid models, custom variables, and responsive media queries)

## 🔧 Installation & Local Development

Follow these steps to run the portfolio locally on your machine:

1.  **Clone the Repository**:
    ```bash
    git clone https://github.com/Tamilselvan-ks-077/Tamil-Portfolio.git
    cd Tamil-Portfolio
    ```

2.  **Install Dependencies**:
    ```bash
    npm install
    ```

3.  **Run Development Server**:
    ```bash
    npm run dev
    ```
    Open `http://localhost:5173` (or the terminal-provided address) in your browser.

4.  **Build for Production**:
    ```bash
    npm run build
    ```
    This generates an optimized bundle in the `dist` directory.
